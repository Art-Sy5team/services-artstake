### Sinkronisasi Status

Untuk mempercepat proses sinkronisasi, ikuti langkah-langkah berikut:

1. **Berhenti Layanan Uptickd:**
   ```js copy
   sudo systemctl stop uptickd
   ```

2. **Cadangkan Status Validasi:**
   ```js copy
   cp $HOME/.uptickd/data/priv_validator_state.json $HOME/.uptickd/priv_validator_state.json.backup
   ```

3. **Unduh dan Terapkan Snapshot:**
   ```js copy
   rm -rf $HOME/.uptickd/data 
   curl https://testnet-files.artstake.net/uptick/snap_uptick.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.uptickd
   mv $HOME/.uptickd/priv_validator_state.json.backup $HOME/.uptickd/data/priv_validator_state.json
   ```

4. **Restart Layanan Uptickd:**
   ```js copy
   sudo systemctl restart uptickd && sudo journalctl -u uptickd -f
   ```
